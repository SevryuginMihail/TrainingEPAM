1.	Одним стримом сгенерировать коллекцию с 10000 рандомными элементами UUID.
2.	Записать в файл.
3.	Одним стримом считать этот файл и посчитать количество элементов UUID, в
    которых сумма цифр > 100
4.	Найти дату конца света по формуле: сегодня + N месяцев + M дней, где N –
    первые два числа от полученного значения, а М – вторые. Значение с ведущими
    нулями, если цифр меньше 4. По тихоокеанской временной зоне. Дату вывести в
    формате ISO с датой и временем (см DateTimeFormatter)
5.	* Создать POJO класс описывающий характеристики колбасы:
string type
int weight
long cost
6.	* Одним стримом считать строки из файла и создать из полученных данных
    экземпляры объектов с данными.
Подсказка: base64
** Создать программу, которая будет составлять объектное представление файловой
    системы. Т.е. на выходе должно получиться дерево объектов, где каждый из
    объектов представляет собой папку/файл с метаинформацией о ней (имя, размер
    и т.д.). Данное дерево должно выводиться в консоли в удобочитаемой форме.
Условия:
•	Использовать изученные по теме инструменты.
•	Никаких намеков на NPE
•	Никаких прямых инициализаций коллекций в задачах без * (не использовать
    конструкции с new)
•	Никаких устаревших API для работы с датой и временем
•	Никаких for и foreach
Критерии оценивания:
До 2 баллов за своевременное и самостоятельную сдачу задания
До 2 баллов за клин код
До 3 баллов за выполнение задания.
До 3 баллов за задания с *.
