На основе базы данных dell-store из прошлого занятия создайте консольное приложение,
которое будет создавать, изменять, просматривать и удалять записи из таблицы products.

Требования:
Запрос на просмотр должен предусматривать просмотр всех записей и искать по полю title
Запрос на изменение должен осуществлятся по уникальному идентификатору таблицы и должен менять поле title
Запрос на удаление записей должен осуществлятся по уникальному идентификатору
Должна быть возможность добавить новые данные
Используйте prepareStatement
В приложении должно быть простое консольное меню
